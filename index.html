// Background

Grounded in Real-World IT

Delien grew out of years of hands-on experience managing servers, networks, and broadcast infrastructure across Papua New Guinea's public and private sectors. Our founders come from environments where uptime isn't optional and troubleshooting needs to happen fast.

We understand PNG's unique infrastructure constraints — unreliable power, limited connectivity in provinces, and a shortage of local IT expertise. We design solutions that work within those realities, not around them.

Whether you need a single subnet planned, a fleet of VMs deployed, or your entire office network rebuilt — we deliver with precision and keep things simple enough for your team to manage long-term.

// What We Offer

Services & Solutions

Specialist IT services for businesses, government, and institutions across Papua New Guinea.

Virtual Machine & Subnetting

Full VM environment design — from hypervisor setup to segmented subnets and inter-VLAN routing

  • Hypervisor installation (KVM, VMware ESXi, Proxmox)
  • Subnet planning and IP scheme design
  • VLAN segmentation and trunk configuration
  • Inter-VM routing and firewall rules
  • VM snapshot, backup, and live migration
  • Resource allocation and performance tuning

IP Address Management (IPAM)

Deploy, organise, and audit every IP across your network — no more spreadsheets

  • PHPIPAM or NetBox deployment and configuration
  • IP allocation policies and documentation
  • DHCP scope management and reservation
  • DNS forward/reverse zone integration
  • IP audit and conflict detection
  • Staff training and handover documentation

Website Building & Hosting

Professional websites with full setup — domain, hosting, CDN, and ongoing care

  • Custom-coded HTML / CSS / JavaScript websites
  • WordPress setup, theming, and plugins
  • GoDaddy / cPanel hosting configuration
  • Cloudflare DNS, CDN, and WAF setup
  • SSL certificate installation and renewal
  • Monthly maintenance, backups, and updates

Server Administration

Windows Server and Linux — from initial setup to day-to-day management and hardening

  • Windows Server 2016 / 2019 / 2022 setup
  • Active Directory, DNS, DHCP, and Group Policy
  • Linux server deployment and hardening (Ubuntu/RHEL)
  • Patch management and update scheduling
  • Remote monitoring and alerting setup
  • Backup and disaster recovery planning

Network Administration

Design, configure, and maintain your LAN/WAN — routers, switches, firewalls, and VPNs

  • Cisco switch and router configuration
  • Firewall rule design and regular auditing
  • Site-to-site and remote access VPN setup
  • Network monitoring (PRTG, Zabbix, Nagios)
  • Wireless access point deployment and tuning
  • Full network documentation and topology diagrams

IT Security & Incident Response

Assess, harden, and respond — keeping your infrastructure secure and compliant

  • Server and network security assessments
  • Malware investigation and remediation
  • Log monitoring and anomaly detection
  • Firewall and access control hardening
  • User privilege and account auditing
  • Security policy and procedure documentation
